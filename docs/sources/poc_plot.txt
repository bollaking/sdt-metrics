Producing plot from :class:`SDT` Object
----------------------------------------

    >>> from sdt_metrics import HI,MI,CR,FA, SDT
    >>> from random import choice
    >>> sdt_obj = SDT([choice([HI,MI,CR,FA]) for i in xrange(1000)])
    >>> print(sdt_obj)
    SDT(HI=251, MI=245, CR=264, FA=240)
